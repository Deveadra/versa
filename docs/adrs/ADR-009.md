# ADR-009: Redesign architecture baseline and contract-first workstream model
Status: Accepted

The redesign baseline is contract-first: shared TypeScript contracts and architecture docs are the authority for cross-workstream integration before deeper runtime implementation.

WS0 defines package ownership, import direction rules, naming standards, and stable contract surfaces for identity/doctrine, memory gateway, workspace state, skills, environment twin, approvals, telemetry, and MCP registry.
